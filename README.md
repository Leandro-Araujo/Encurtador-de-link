Encurtador-de-link
==================

Pequeno website feito em node.js

Digite o site, sem usar o http://
Ele vai gerar na frente o código, pegue-o, e o coloque depois da /


Instalação
==========

```
git clone <REPO_URL>
cd Encurtador-de-link
npm install
```

Executando
==========

```
node app.js
```
